# Global Real Estate Agent (Portfolio)

A fictional global real estate agent website built as a portfolio project. Single-page site with hero, about, services, property listings, markets, testimonials, and contact.

**Live:** [agent.nicolae-valentin-dinca.net](https://agent.nicolae-valentin-dinca.net)

## Tech stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- Static content, no backend

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Project note

This is a **fictional agent & portfolio project**. No real transactions or personal data are involved. Portfolio of [Nicolae-Valentin Dinca](https://nicolae-valentin-dinca.net).
